# 逆命蛊途 · 网页试玩版

原创暗黑东方蛊修卡牌肉鸽 Demo。纯静态 HTML/CSS/JS，无需安装。

## 试玩链接

**正式入口（固定不变，发这一个给玩家）**：

https://kazezefwefew.github.io/inverse-gu-path-web/

- 玩家始终使用上面这个固定链接，不需要每次更换链接。
- 看到旧版本时，刷新页面（手机可下拉刷新或清缓存重开）即可加载最新版。
- 开发者测试新版时，可临时在链接后加 `?build=v版本号`（例如 `?build=v0.9.2.5`）强制刷新缓存。它**不会锁定旧版本**，页面始终加载最新资源；请勿把带 `?build=` 的链接当正式入口长期分发。
- 手机玩家打开后请横屏游玩，竖屏会显示横屏提示。

当前发布版本：V0.9.2.6。
